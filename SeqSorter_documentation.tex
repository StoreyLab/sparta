\section{SeqSorter}
SeqSorter is a python script designed to disentangle two SAM files that map the same RNAseq reads to different genomes.\\

\noindent
\textbf{Requirements:}\newline
Python 2.7.7 (Python 3.4 compatibility possibly to come) \newline
Samtools (\verb https://github.com/samtools/samtools )\newline
Pysam (\verb https://github.com/pysam-developers/pysam )
